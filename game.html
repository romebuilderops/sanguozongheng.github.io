三国纵横消战天下

一款融合消除玩法和三国题材的策略游戏

在浏览器中打开游戏

如何运行游戏

由于环境限制，建议按照以下步骤在本地运行游戏：

  • 1. 确保安装了 Node.js（版本 16.13.1 或更高）
  • 2. 在项目根目录执行：npm install
  • 3. 启动开发服务器：npm run dev:local
  • 4. 在浏览器中访问：http://localhost:5173

游戏特色

  • • 阵营选择：魏、蜀、吴、群雄
  • • 将领系统：包含主公和副将
  • • 技能系统：每个将领都有独特技能
  • • 战斗系统：消除类玩法
  • • 关卡系统：挑战不同难度的关卡
  • • 美术风格：水墨+国潮Q版
