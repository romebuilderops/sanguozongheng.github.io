# 需求文档

## 1. 应用概述

- **应用名称**：三国纵横·消战天下
- **应用描述**：一款融合三消玩法与卡牌将领养成的策略手游。玩家扮演乱世主公，通过三消战斗参与三国历史战役，消除棋子产出资源，养成将领，推进主线剧情最终一统天下。

---

## 2. 用户与使用场景

### 2.1 目标用户
- 年龄 18-35 岁，喜欢策略游戏、三国题材及消除类休闲游戏
- 游戏时间碎片化，单局 5-15 分钟
- 有收集养成倾向，喜欢阵容搭配和资源管理

### 2.2 核心使用场景
- 挑战战役关卡（三消战斗）→ 获得资源 → 养成将领 → 挑战更强关卡（循环）
- 单机或异步竞争，不强制实时 PVP

### 3.2 主菜单页
- **核心功能**：
  - 展示游戏 Logo 及动态背景动画（三国战场风格）
  - 提供「开始游戏」按钮，点击直接进入战役关卡选择页
  - 提供「设置」按钮，点击弹出设置面板
  - 提供「退出」按钮，点击确认后退出游戏
  - 提供存档的注册起名功能
- **设置面板**：
  - 音乐开关（默认开启）
  - 音效开关（默认开启）
  - 音量调节滑块（0-100）

### 3.3 底部导航栏
- **功能**：在战役、将领、抽卡、商店页面之间切换
- **导航项**：
  - 战役：进入战役关卡选择页
  - 将领：进入将领页
  - 抽卡：进入抽卡页
  - 商店：进入商店页
### 3.4 战役关卡选择页
- **章节列表**：按三国编年史顺序排列
  - 序章：黄巾之乱（1-10关）

  - 第二章：官渡之战（21-35关）
  - 第三章：赤壁之战（36-50关）
  - 第四章：三分天下（51-70关）

### 3.5 战斗页（三消核心）

#### 3.5.1 顶部 HUD
- 主公头像、等级、主公名称
- 当前生命值（数值 + 进度条）
- 当前能量值（数值 + 进度条，上限 100）
- 设置按钮（暂停 / 退出关卡）

#### 3.5.2 8×8 三消棋盘
- **棋盘规格**：8×8 网格，竹简卷轴风格边框
- **棋子类型（6种基础色）**：
  - 红色（攻击）：消除时对敌方造成伤害
  - 绿色（生命）：消除时回复我方生命
  - 黄色（士气）：消除时增加技能能量
  - 蓝色（防御）：消除时获得临时护盾
  - 紫色（谋略）：消除时增加连击点数
  - 橙色（粮草）：消除时掉落额外资源
- **特殊棋子**：
  - 强化棋子：4个连珠生成，消除时触发范围效果
  - 炸弹棋子：5个连珠生成，消除时炸毁 3×3 区域
  - 十字消除棋子：T/L 型消除生成，消除时触发十字范围消除
- **操作方式**：
  - 滑动交换：拖动相邻棋子交换位置；交换后若能形成 ≥3 个同色连珠则生效，否则无效并播放错误提示，交换动画时长 0.1 秒
  - 消除判定：交换完成后 0.05 秒进行消除判定
  - 长按棋子：显示该棋子类型说明
- **消除规则**：
  - ≥3 个同色棋子直线相连（横/竖）触发消除
  - 4个连珠：生成强化棋子
  - 5个连珠：生成炸弹棋子
  - T/L 型消除：生成十字消除棋子
- **伤害计算**：
  - 单次消除伤害 = 基础攻击力 × 消除数量系数 × 连击加成
  - 基础攻击力 = 主公攻击力 + 当前场上将领攻击力之和 × 0.3
  - 消除数量系数：3个=1.0，4个=1.5，5个=2.0，6个以上=2.5
  - 连击加成：每连击1次 +5%，上限 100%
- **回合机制**：每次玩家消除后，敌人进行一次攻击；无时间限制，回合制

#### 3.5.4 敌方信息栏
- 展示敌方名称、等级、当前生命值（数值 + 进度条）

  - 展示获得奖励（银两、木材、将领碎片、经验书）

  - 展示失败原因
  - 提供「再次挑战」「返回」按钮

### 3.6 将领页
- **将领列表**：
  - 展示已拥有将领列表，含头像、名称、阵营、稀有度、当前等级/星级
  - 支持按阵营、稀有度、等级排序
- **将领详情页**：
  - **基础属性**：攻击、生命
  - **升级功能**：消耗银两和将领经验书，提升攻击/生命属性
  - **升星功能**：消耗将领碎片、银两和经验书，提升将领品质和属性，最高 5 星
  - **技能槽**：初始解锁 1 个，升星后解锁更多（每个将领最多 3 个技能槽）

### 3.7 抽卡页
- **卡池选择**：普通池 / 限定池（限时活动）
- **抽卡方式**：
  - 单抽：消耗100元宝
  - 十连抽：消耗900元宝（9折优惠）
- **抽卡结果**：
  - 动画演出（卡牌翻转效果）
  - 获得将领碎片或道具列表
  - 十连抽必出至少1个稀有及以上品质将领

### 3.8 商店页
- **商品分类**：
  - 体力药水：恢复体力
  - 银两袋：增加银两
  - 木材包：增加木材
  - 将领经验书：用于将领升级
  - 万能碎片：可用于任何将领升星

- **购买规则**：
  - 消耗元宝购买
  - 部分道具每日限购数量限制
  - 首充礼包：首次充值额外赠送稀有将领

### 3.9 任务页
- **每日任务**：
  - 展示任务列表及完成进度
  - 完成任务可领取元宝、银两、经验书等奖励
- **新手7日登录奖励**：
  - 每日登录领取对应奖励
  - 展示倒计时
  - 累计登录7天可获得传说将领
- **成就系统**：
  - 完成特定目标获得成就点数
  - 解锁成就奖励

### 3.10 背包页
- **道具列表**：
  - 展示当前持有道具列表（类型、数量）
  - 支持按类型筛选
- **道具操作**：
  - 点击道具可查看详细说明
  - 部分道具可直接使用（如体力药水）
  - 可批量使用道具

### 3.11 关卡结算页
- **胜利结算**：
  - 星级评定：根据剩余生命值、回合数、技能使用次数综合评定
  - 奖励发放：银两、木材、将领碎片、经验书
  - 额外奖励：首次通关额外奖励
- **失败结算**：
  - 失败原因：生命值归零、超时（限制回合关卡）
  - 消耗体力：挑战失败不消耗体力
  - 重新挑战：点击按钮直接重新进入战斗

### 4.5 体力系统
- 初始体力：100
- 战役关卡消耗：10体力/次
- 体力恢复：5分钟恢复1点
- 体力上限可通过升级主殿提高，最高 200
- 体力不足时提示购买体力药水或等待恢复
- 每日体力购买次数限制：5次（随VIP等级提升）

### 4.6 成长数值
- 主公升级经验 = 100 × (当前等级^1.2)
- 将领升级经验 = 50 × (当前等级^1.3)
- 主公每升一级：体力上限 +2，解锁新功能/建筑
- 将领每升一级：攻击 +5，生命 +50（基础值）
- 将领升星成长率：每升一星，属性成长率提升10%

### 4.7 敌人 AI 行为
- 敌人类型：普通兵种（步兵/弓兵/骑兵）、将领BOSS、守城器械
- 普通兵种：每回合对玩家造成固定伤害（10-50）
- 将领BOSS：每3回合释放一次全体攻击技能
- 守城器械：每回合减少玩家能量或随机消除棋子
- 行为逻辑：

## 5. 异常与边界情况

| 场景 | 异常情况 | 处理方式 |
|---|---|---|
| 棋子交换 | 交换后无法形成 ≥3 连珠 | 交换无效，播放错误提示动画，棋子复位 |
| 棋盘死局 | 棋盘上无任何可消除组合 | 自动重新随机排列棋盘，不消耗回合 |
| 技能释放 | 能量不足 | 按钮置灰，点击时提示「能量不足」 |
| 体力不足 | 挑战关卡时体力不足 | 弹窗提示，提供「购买体力药水」或「等待恢复」选项 |
| 玩家生命归零 | 战斗中生命值降至0 | 触发失败结算，展示失败页面 |
| 网络中断 | 战斗中断网 | 本地保存当前战斗状态，重连后可继续 |
| 升级资源不足 | 银两或木材不足时点击升级 | 提示「资源不足」，引导前往商店或挑战关卡 |
| 将领碎片不足 | 升星时碎片不足 | 提示「碎片不足」，展示碎片获取途径 |
| 元宝不足 | 购买道具或抽卡时元宝不足 | 提示「元宝不足」，引导前往充值界面 |
| 每日购买限制 | 超过每日体力购买次数限制 | 提示「购买次数已达上限」 |
| 关卡未解锁 | 尝试进入未解锁的关卡 | 提示「关卡未解锁」，引导完成前置关卡 |
| 建筑等级上限 | 尝试升级已达最高等级的建筑 | 提示「建筑已达最高等级」 |
| 将领等级上限 | 尝试升级已达最高等级的将领 | 提示「将领已达当前等级上限」 |
| 抽卡次数限制 | 超过每日抽卡次数限制 | 提示「抽卡次数已达上限」 |
| 任务已完成 | 尝试重复领取已完成的任务奖励 | 提示「任务奖励已领取」 |
| 登录奖励已领取 | 尝试重复领取当日登录奖励 | 提示「今日奖励已领取」 |
